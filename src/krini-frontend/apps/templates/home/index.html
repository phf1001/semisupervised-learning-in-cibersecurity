Analizador de phishing

¡Bienvenido al analizador de enlaces de phishing! Para utilizarlo, tan sólo escribe la dirección de la página que quieres analizar y pulsa el botón "Scan". En el dashboard podrás observar algunas de las gráficas generadas por nuestros estimadores.

¿Cómo funciona?

El resultado se obtiene de generar un vector de características asociado a la página que se introduzca. Una vez conseguido, se procesa mediante algoritmos de Machine Learning para obtener un resultado que nos indique si la página es o no phishing.

¿Puedo denunciar enlaces?

¡Por supuesto! Además de utilizar algoritmos de aprendizaje semisupervisado, también puedes denunciar enlaces sospechosos para que sean analizados por nuestro equipo de expertos. Tan sólo tienes que iniciar sesión.
